# Python Calculator

A simple command-line calculator built with Python.

## Folder Structure

The calculator program is located inside the **`Calculator`** folder.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Modulus
* Power

## Concepts Practiced

* Functions
* Dictionaries
* User Input
* Conditional Statements (`if`, `elif`, `else`)
* Arithmetic Operators
* Function Calls

## How to Run

```bash
python Calculator.py
```

> Make sure you are inside the **Calculator** folder before running the program.
